# Subheadline

Displays the subheadline of the story in the story card.

## Usage

```jsx
<Subheadline story={story} truncateChars={200} />
```

<!-- PROPS -->

We can pass `showFullContent` as props to show the complete sunheadline without truncating.

```jsx
<Subheadline story={story} truncateChars={200} />
```
